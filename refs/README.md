# Reference submodules

These submodules provide source references for implementation and parity work.

- `refs/subforge` -> https://github.com/wiedymi/subforge.git
- `refs/text-shaper` -> https://github.com/wiedymi/text-shaper.git
- `refs/libass` -> https://github.com/libass/libass.git

Initialize on a new machine:
```
git submodule update --init --recursive
```
